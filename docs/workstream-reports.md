# Workstream Reports

`REPORT.md` is required before `work complete`.

## Workflow

```bash
work report init       # create template if missing
work report validate   # enforce required sections
work complete          # requires valid REPORT.md
```

## Required Content

- Summary
- Accomplishments (by stage)
- File references
- Issues/blockers
- Next steps

## Metrics Helpers

```bash
work report metrics
work report metrics --blockers
work report --stage 1
```
